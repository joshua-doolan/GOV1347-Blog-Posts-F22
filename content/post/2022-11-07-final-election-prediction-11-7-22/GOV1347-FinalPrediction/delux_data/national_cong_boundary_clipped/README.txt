National Congressional Districts for 2022 (10/17/2022) Clipped to Cartographic Boundaries

## RDH Creation Date
10/31/2022

## Sources
Sourcing for Congressional district plans varies by state and are the same sourcing as the Congressional District plans by state on Redistricting Data Hub state pages (e.g. https://redistrictingdatahub.org/state/alabama/). 
Most files are titled with "Adopted Congressional Plan" and listed under Featured Datasets. Refer to metadata and READMEs of these files for additional sourcing information.

Cartographic Map from the [US Census Bureau](https://www.census.gov/geographies/mapping-files/time-series/geo/carto-boundary-file.html)

## Fields
DISTRICT		District Number (AT-LARGE for states w/ one representative)
STATE		State Abbreviation

## Processing Notes
District shapefiles were concatenated together using the pandas and geopandas libraries in Python and set to the same CRS.
Files were cleaned and organized to have uniform fields.
Data was clipped to Cartographic Boundary file listed in source.
Data was extracted as a single shapefile (.shp).

## Additional Notes
The assignments in this file reflect what are, to the best of our knowledge, the current Congressional districts.
